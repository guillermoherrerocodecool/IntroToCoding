Modules

Months 1-3
Prog Basics

The first steps of your programming journey will be done in Python. This friendly programming language is used to create web applications, data analysis and machine learning. Additionally, you will practice how to write clean code, improve your problem solving skills and how to use the GIT version control tool.

Months 4-6
Web

Here you start creating web applications, learning basic use of databases and SQL; and many more things related to front-end technologies, like HTML, CSS and Javascript. Additionally you will also build up your knowledge of giving presentations, SCRUM and project management.

Months 7-9
OOP

Here you will dive in into the Java programming language and into much more complex applications. Practicing the fundamentals of object oriented programming and the SOLID principles is key in this module. Additionally, you will increase your code quality with testing and extend your knowledge of databases and SQL.

Months 10-12
Advanced

We prepare you for future job interviews. Complex topics will become part of your daily routine, like multithreading, data structures and transactions in SQL. You will learn a back-end framework - Spring or ASP.NET. You will specialize yourself further with other technologies, like Java, C# or PHP, React, DevOps and automated testing.
